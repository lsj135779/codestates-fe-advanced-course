- 프로젝트 실행 방법
  - git clone을 통해 파일을 받습니다.
  - codestates-fe-advanced-course 폴더로 이동합니다.
  - npm install (npm i) 를 통해 사용한 모듈들을 설치합니다.
  - npm start 로 페이지를 띄웁니다.
